# Node-typescript-express-boilerplate
This repo is a  Node JS Typescript boilerplate that uses Express, Mongoose, Eslint, Prettier and Husky.

## Installation

Use the package manager [npm](https://npmjs.com).

```bash
npm install
```

## Usage

```bash
nodemon
```
